### project goal
Create web app where users can find and upload alternative images of existing magic the gathering cards.

### values
simplicity:
- less code is better, minimize comments, use concise & descriptive names, maximize readability for humans
- minimize number of steps/clicks for users to achieve their goal

### user journeys
1. type search query
2. select card from results
3. upvote/downvote alternatives
4a. download high res image of selected alternative
4b. upload new high res image

### hints
- search using scryfall.com api
