# feature overview
- search query input textfield/input
- image gallery showing cards fitting to user provided query
- image gallery showing alternatives to a card that other users have uploaded

# user flow
1. submit query -> show results (images) from scryfall (https://api.scryfall.com/cards/search?q=<query>)
2. select card from results -> show alternatives
3. optional upload new alternative (image) -> new alternative appears along with the others
